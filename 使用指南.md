# 答辩材料生成器 - 使用指南

## 🌐 访问地址

```
http://172.16.78.4:8888
```

**重要提示**: 
- 请确保你的电脑与服务器在同一局域网（同一WiFi）
- 如果无法访问，请联系管理员

---

## 📋 使用步骤

### 1. 打开浏览器
在地址栏输入: `http://172.16.78.4:8888`

### 2. 选择平台
- **北京快手** - 北京快手科技有限公司
- **成都快购** - 成都快购科技有限公司

### 3. 填写案件信息

**必填项**:
- 原告
- 被告  
- 案号（例如：(2026)京73民初277号）
- 店铺名
- 受理法院

**选填项**:
- 订单编号（如果没有会自动删除）

### 4. 生成文档
点击"生成答辩材料"按钮，系统会自动生成并下载3个文档：
1. 📄 授权委托书
2. 📄 任职证明
3. 📄 答辩状

## 🐛 常见问题

**Q: 无法访问页面？**  
A: 检查是否在同一WiFi，或联系管理员确认服务是否运行

**Q: 生成失败？**  
A: 检查必填项是否都已填写，刷新页面重试

**Q: 文档格式不对？**  
A: 检查输入的案号、法院等信息格式是否正确
